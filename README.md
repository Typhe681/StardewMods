This repository contains my SMAPI mods for Stardew Valley.
## Mods
* **Snowballs** <small>([Nexus](https://nexusmods.com/stardewvalley/mods/42769))</small> 
_Dig up snowballs and shoot them at NPCs, pets, and monsters using a slingshot._

* **Minecart Issues** <small>([Nexus](https://www.nexusmods.com/stardewvalley/mods/43079))</small> 
_Nerfs the minecart system. Minecarts will occasionally break down._
